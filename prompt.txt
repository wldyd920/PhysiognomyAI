내가 하고자 하는 것은 관상 통계 웹사이트인데 웹 어플리케이션이면서 데이터 분석도 하면서 API 서비스도 하는 서비스야. 주요 기능은 다음과 같아.


주요 기능 :
1. 기본적인 기능은 유저가 자신의 얼굴 이미지를 올리면 관상학적으로 분석해주는거야. 텍스트 위주로 하는데, 주의해야 할 점은, 현대의 fortunteller나 관상가, 점술가들이 말하듯이 설득되는 톤으로 해야되고, 긍정적인 측면만 말하면 설득력이 떨어지겠지? 그럴싸하게 말해야돼.

2. 이걸 구현하기 위해서 내 chat GPT 계정을 이용해서 이미지를 분석해달라는 요청을 할거야. 아마 assistant로 어떤 작업을 원하는지 미리 말해놓고 사용하는 방법도 있을 수 있겠지.

3. 내 chat GPT계정을 이용하려면, 웹사이트에서 받은 이미지를 API 호출 가능한 곳으로 보내는 방법을 사용해야 할거야. 왠만하면 Github Pages에서 구현을 하고싶지만, 어려울거 같아서 Replit에서 구현하고자 해. 이게 Replit에서 가능한지도 알고싶어.

구현 방법 :
1. 먼저, 가장 쉽게 구현할 수 있는 방법은 chatGPT로 입력된 이미지를 관상학적으로 분석한 뒤에, 웹 사이트에 결과를 올려주는거야.

2. 조금 더 어렵게 하자면, pre-trained model을 다운로드 받아서 입력된 이미지로 test만 거쳐서 Big-Five(외향성 (Extraversion), 호감성 (Agreeableness), 성실성 (Conscientiousness), 신경증 (Neuroticism), 경험에 대한 개방성 (Openness)) 성격을 알려주는 거야. 이 경우에는 과연 chatGPT보다 pre-trained model이 더욱 성능이 좋은지 확인을 거쳐야해. 이 방법의 장점으로는 dinamic website가 아니어도 된다는거야.

3. 조금 더 어렵게 하자면, 모델을 직접 설계하는거야. Big-Five 중에서도 각각 하나씩 실수나 정수형태로 점수를 매겨주는 모델을 5개 따로 train해서 각각의 점수를 매기는 것이야. 이 경우에는 데이터세트가 확실히 있어야돼. 사용 가능한 모든 오픈소스 데이터세트를 사용해도 부족할 수도 있어. 그리고 CNN이 아니라 attention이나, diffusion모델을 사용하면 더욱 효과가 좋겠지만 CNN을 반드시 섞어야 할수도 있어. 왜냐하면, 관상학적으로 얼굴의 어느 부위가 어떤 성격에 어떻게 영향을 미쳤는지 설명하는게 관상학이라고 할 수 있으니까 말이야.

4. 조금 더 어렵게 하자면, 데이터세트를 추가적으로 자체적으로 만드는거야. 고품질 데이터세트는 아니겠지만, 자동화를 통해서 데이터세트를 만들면 시간을 절약하면서도 큰 데이터세트를 만들 수 있을거라 생각해. 예를 들자면, 유튜브나 인플루언서들의 성격을 검색하거나 동영상에서 따와서 label을 만들고, 해당 인플루언서의 다양한 표정 등을 검색하거나 유튜브의 동영상에서 찾아와서 data로 사용하는거야. 이렇게 하면 대부분의 데이터 세트가 외향형으로 편향될 가능성이 있긴 하겠지만, 그 외의 성격을 판단하기에는 좋은 수단이 될 수도 있을거라고 생각해.

5. 조금 더 어렵게 하자면, 사람들이 올린 사진을 데이터세트로 사용하는거야. 사용자들이 얼굴 사진을 올려서 사용한 후에 리뷰를 받는거지. 예를들어, 실제 자신의 성격과 일치했는지 안했는지를 표시하게 하는거야. 아니면, 자신의 MBTI를 적게 한다던지, MBTI 테스트를 간단히 실행하게 할 수 있게 한다던지 하는거지. 그리고 그 결과를 model에 추가적으로 train시켜서 parameter를 수정하는거야. 이게 성공적으로 실행되고 사용자가 많아진다면 우리 모델은 점점 더 정확도가 올라갈거야.
